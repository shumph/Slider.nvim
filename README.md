# Slider

This is a presentation plugin for nvim, written in lua

# Usage
```lua
require("slider").start_presentation {}
```

Use n to move to the next slide, p to the previous one, and q to quit

# Installation

### [lazy.nvim](https://github.com/folke/lazy.nvim)
```lua
{
 "shumph/slider.nvim"
}
```
